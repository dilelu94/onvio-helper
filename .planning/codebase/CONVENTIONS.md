# Coding Conventions

- **Mixed Tech Stack**: Development involves Shell (Bash), Python, and JavaScript/TypeScript.
- **Naming**: Scripts use lowercase with underscores (e.g., `create_html_bookmarks.py`, `toggle_lights.sh`).
- **Configuration**: JSON-based configuration (`.sudvision-config.json`) and Windows-style `.reg` files for some components.
- **Organization**: Home directory contains loose scripts alongside specialized directories (`scripts/`, `repos/`).
- **Automation**: Heavy reliance on cron or manual script execution for system management.
