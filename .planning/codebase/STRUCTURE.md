# Directory Structure

- **.config/**: Application-specific configuration files.
- **.local/**: User-specific data and binaries.
- **scripts/**: Centralized location for system automation scripts.
- **repos/**: Development projects and source code repositories.
- **Applications/**: User-installed software applications.
- **.planning/**: Project-specific analysis and planning documents.
- **.gemini/**, **.gsd/**: Agent-specific configuration and state.
- **AppImages/**: Standalone application packages.
