# External Integrations

- **Google Cloud Platform**: GCloud SDK usage for cloud management.
- **Gaming Platforms**: Steam integration (`.steam/`, `.steampath`), Lutris configurations (`kin-and-quarry-lutris.yml`).
- **Hardware/Peripherals**: OpenRGB for lighting control (`openrgb-udev-install.sh`), HeadsetControl.
- **System Automation**: Chrome/Tab management (`create_html_bookmarks.py`, `extract_tabs.py`, `recover_older_tabs.py`).
- **Shell Customization**: Custom Bash configurations (`.bashrc`, `.bashrc.backup`).
