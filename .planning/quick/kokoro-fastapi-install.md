# Quick Task: Install Kokoro-FastAPI

## Goal
Install Kokoro-FastAPI in ~/repos/Kokoro-FastAPI inside the 'cuda-env' distrobox with CUDA support.

## Success Criteria
- [ ] Repo cloned to ~/repos/Kokoro-FastAPI.
- [ ] 'espeak-ng' installed inside 'cuda-env'.
- [ ] 'uv' environment setup and dependencies installed.
- [ ] Models downloaded using 'docker/scripts/download_model.py'.
- [ ] Start script configured for quick execution.

## Tasks
- [ ] Task 1: Clone Kokoro-FastAPI repository.
- [ ] Task 2: Install system dependencies (espeak-ng) in 'cuda-env'.
- [ ] Task 3: Create venv and install python dependencies with 'uv'.
- [ ] Task 4: Download models and configure start script.
