# Quick Task: Install Kokoro-FastAPI

## Goal
Install Kokoro-FastAPI in ~/repos/Kokoro-FastAPI inside the 'cuda-env' distrobox with CUDA support.

## Success Criteria
- [x] Repo cloned to ~/repos/Kokoro-FastAPI.
- [x] 'espeak-ng' installed inside 'cuda-env'.
- [x] 'uv' environment setup and dependencies installed.
- [x] Models downloaded using 'docker/scripts/download_model.py'.
- [x] Start script configured for quick execution.

## Tasks
- [x] Task 1: Clone Kokoro-FastAPI repository.
- [x] Task 2: Install system dependencies (espeak-ng) in 'cuda-env'.
- [x] Task 3: Create venv and install python dependencies with 'uv'.
- [x] Task 4: Download models and configure start script.
