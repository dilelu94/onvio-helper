  <action>
    Find the latest x86_64 (amd64) AppImage download URL from pot-app/pot-desktop releases using the GitHub API. 
    Download the file into ~/AppImages/ and make it executable.
    
    Command sequence:
    1. URL=$(curl -s https://api.github.com/repos/pot-app/pot-desktop/releases/latest | grep "browser_download_url" | grep "amd64.AppImage" | head -n 1 | cut -d '"' -f 4)

    3. chmod +x /var/home/dilelu/AppImages/$(basename $URL)
  </action>
