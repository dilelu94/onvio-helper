# Project: onvio-helper

## Core Value
Electron + React desktop application to automate repetitive tasks in the Onvio platform using Playwright and Python, providing a user-friendly interface for configuration and script execution with live logging.

## Core Pillars
- **Ease of Use**: GUI for credential and company management (no hardcoded credentials).
- **Modularity**: Each script/functionality isolated in its own file.
- **Reliability**: Strict TDD approach (one test per functionality).
- **Transparency**: Real-time script execution logs in the UI.

## Constraints
- Platform: Windows (Electron)
- Frontend: React
- Automation: Playwright / Python
- Testing: Strict TDD (one test per functionality)
- Distribution: Local use for accountants/consultants
